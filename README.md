## Desafio Poo Dio
> O código representa o projeto feito através do curso de POO da Dio.

## 🚀 Tecnologias utilizadas
- Java 17 (ou Java 8+)
- Programação Orientada a Objetos
- Collections Framework
- PI de Datas (java.time)

## ▶️ Como executar
1. Compile os arquivos
- javac *.java
2. Execute a classe principal
- java Main

Ou execute pela IDE:

- IntelliJ
- Eclipse
- VS Code
- NetBeans
